# Homebrew TBLang

Homebrew tap for [TBLang](https://github.com/SwanHtetAungPhyo/tblang) - Plugin-based Infrastructure as Code language.

## Installation

```bash
brew tap swanhtetaungphyo/tblang
brew install tblang
```

## Usage

```bash
tblang --version
tblang plugins list
tblang plan infrastructure.tbl
tblang apply infrastructure.tbl
```
